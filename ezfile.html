Example 1. - 하나만 업로드 가능
$(function(){ $("#file1").ezfile({name: "upfile1"}); })
Example 2. - 무제한 업로드 가능
$(function(){ $("#file2").ezfile({name: "upfile2", limit: 0}); })
Example 3. - 업로드 3개 이하로 제한
$(function(){ $("#file3").ezfile({name: "upfile2", limit: 3}); })
Example 4. - 첨부 가능 확장자명(gif, jpg, jpeg, png)
$(function(){ $("#file4").ezfile({name: "upfile4", ext: ["gif", "jpg", "jpeg", "png"]}); })
Example 5. - 첨부 가능 확장자명(gif, jpg, jpeg, png), 이하(900px, 300px) 사이즈 허용
$(function(){ $("#file5").ezfile({name: "upfile5", ext: ["gif", "jpg", "jpeg", "png"], size: [900, 300, true]}); })
Example 6. - 첨부 가능 확장자명(gif, jpg, jpeg, png), 동일(900px, 300px) 사이즈 허용
$(function(){ $("#file6").ezfile({name: "upfile6", ext: ["gif", "jpg", "jpeg", "png"], size: [900, 300]}); })
Example 7. - 첨부 가능 확장자명 gif 하나만, 동일(8, 2) 비율 허용
$(function(){ $("#file7").ezfile({name: "upfile7", ext: "gif", ratio: [8, 2]}); })
Example 8. - 파일당 용량제한(10mb)
$(function(){ $("#file8").ezfile({name: "upfile8", limit: 0, byte: "10mb"}); })
